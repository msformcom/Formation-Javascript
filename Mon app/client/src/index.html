Mon Application chargée auto 2

  • Installer node.js
  • Dans un terminal : Set-ExecutionPolicy -ExecutionPolicy Unrestricted -Scope CurrentUser
  • npm init : Initialiser le fichier packages.json
  • npm install typescript --save-dev : Installer le compilateur typescript
  • npx tsc --init : Créer le fichier tsconfig.json
  • npx tsc : Compiler
  • npx tsc --watch: Compiler à la volée
